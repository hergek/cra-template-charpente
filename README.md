# Charpente

    The template for your web Application, using ReactJs.

## Installation

with npx

```bash
npx create-react-app [app-name] --template charpente
```

or with npm

```bash
npm create-react-app [app-name] --template charpente
```

or with yarn

```bash
yarn create react-app [app-name] --template charpente
```
